This repository contains the mini-projects, database integrations, and Object-Oriented Programming (OOP) concepts I explored and developed during my second year as a Software Engineering student.

## Repository Structure

### `Python_Mini_Projects`
Functional console applications demonstrating file handling, modules, and database integration.
* **Credit Ledger (`credit_ledger.py`):** A debt tracking system integrated with an **SQLite3** database.
* **Phonebook CLI (`phonebook.py`):** A dictionary-based contact management system.
* **Notepad App (`notebook.py`):** Demonstrates Python's File I/O (Read/Write) capabilities.
* **Raffle App (`raffle.py`):** Uses the `random` module for drawing lots.

### `Python_Learning_Notes`
Comprehensive study notes covering the core mechanics of Python.
* **`python.py`:** Fundamental Python concepts, variables, loops, and control structures.
* **`files.py`:** Introduction to File I/O operations (reading, writing, and appending text files).
* **`modules.py`:** Working with Python's built-in modules and importing external libraries.
* **`oop.py`:** Object-Oriented Programming principles including classes, `__init__` methods, instance attributes, and Inheritance (`super()`).
* **`database.py`:** Introduction to SQLite3, database connections, and executing basic SQL queries within Python.

> *Note: Some personal learning notes within the `Python_Learning_Notes` folder are kept in my native language (Turkish) to preserve my authentic learning process.*
